    Альт Контроллер

    Операционная система для программируемых логических контроллеров на базе ядра Linux realtime

    Скачать Купить

    «Альт Контроллер» — особый дистрибутив операционной системы семейства Альт, предназначенный для использования в составе с программируемыми логическими контроллерами, в системах где критична скорость реакции на внешние события, важна отказоустойчивость и минималистичность.

    Почему и где  используют «Альт Контроллер»

    ПО входит в реестр Минцифры (регистрационный номер ПО: ????)
    Дистрибутив выпускается в нескольких вариантах исполнения: для x86 (64-битных), AArch64 (Huawei Kunpeng, ThunderX и другие), e2kv5/e2kv6 («Эльбрус»).
    Основан на ядре Linux realtime - ядре, запускающим операционную систему как низкоприоритетный процесс, с приоритетом по выполнению сигналов от сторонних устройств, тем самым минимизируя время отклика на события от высокоприоритетных устройств                     
    Содержит сетевые службы и сервисы для полноценного функционирования корпоративной инфраструктуры
    Содержит сетевые службы и сервисы для полноценного функционирования в  
    Для построения офисной инфраструктуры в рамках импортозамещения рекомендуется использовать вместе с «Альт Рабочей станцией»
    Совместимость с экосистемой ПО отечественных производителей
    Система управления сервером имеет более 100 модулей и позволяет работать через графический или веб-интерфейс

    * Программный комплекс «Альт Домен» лицензируется и поддерживается отдельно.
    ** 

    Технологизация данного процесса предполагает следующие особенности: • специализированную, первичную заказную разработку под конкретные аппаратные изделия; • обобщение и автоматизацию сборки под конкретные виды сопровождаемых аппаратных изделий; • многоступенчатый подход при подготовке продукта, включающий в себя: • регулярный выпуск обновлений первичного "каркасного образа корневой файловой системы" в виде архива (в форматах tar.gz, tar.xz и т.п.) корневой файловой системы или iso-образа (для сертифицированных продуктов), как первоисточника для генерации архива в виде вторичного "каркасного образа корневой файловой системы", с помощью mkimage-профиля; • реализацию инструментария для генерации прошиваемого в конкретное изделие "предустановливаемого варианта образа" в "сыром" формате (как правило, это образ блочного устройства), с помощью alt-rootfs-installer или расширенного функционального аналога; • техническое обеспечение генерации при подготовке прошивки (в "предустановливаемом варианте образа"): • таблицы разделов под целевое изделие; • корректного размещения загрузчика; • взаимносогласованного между загрузчиком и ядром файла с описанием device tree (dtb-файла); • обновляемых штатными средствами ядра и корневой файловой системы; • дополнительного «вендорского» программного обеспечения.  

    Сроки поддержки в части обновлений по безопасности (если иное не предусмотрено условиями поставки):

    • - поддержка 9 платформы завершена 31.12.2023 г., продажи дистрибутивов на 9 платформе прекращены, сертификаты вендорской технической поддержки дистрибутивов на 9 платформе обслуживаются в соответствии с SLA до полного завершения их действия;
    • - для дистрибутивов «Альт Сервер» 10 версий — 31 декабря 2025, но не ранее полугода с момента выпуска новой версии (11.0).

    Сервер антиспама Spamassassin
    Прокси-сервер Squid
    Веб-серверы Apache, NGINX
    Сервер подключений VDI
    Средства для создания зеркала репозитория для централизованного обновления и инструменты разработчика
    Система резервного копирования серверов и рабочих станций под управлением Windows и Linux
    Графическая среда
    Сервер сетевой установки (загрузки)
    Сервер совместной работы Nextcloud
    Сервер мониторинга Zabbix
    Серверы баз данных (СУБД)
    PostgreSQL, MariaDB
    FreeIPA для централизованной аутентификации пользователей
